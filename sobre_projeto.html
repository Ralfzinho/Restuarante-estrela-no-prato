Sobre mim – Ralf da Silva

Minha História

Tenho 22 anos e me chamo Ralf da Silva. Sou estudante de Análise e Desenvolvimento de Sistemas (ADS) pela Universidade de Passo Fundo (UPF), atualmente no segundo semestre Desde criança, sempre fui muito curioso e criativo, com uma imaginação voltada para um mundo super tecnológico — quase "cyber". Sempre me perguntei: como posso transformar esse mundo imaginado em realidade? Foi essa inquietação que despertou meu interesse pela área da tecnologia e me trouxe até aqui

O projeto Restaurante Estrelas no Prato surgiu durante uma aula de Desenvolvimento Web – Nível I inicialmente como um exercício simples para praticar o uso de grids e flexbox. A proposta era básica, mas conforme fui desenvolvendo, novas ideias começaram a surgir e o projeto cresceu junto com meu aprendizado. Embora ainda esteja em estágio inicial, meu objetivo é transformá-lo em um trabalho mais completo e bem estruturado, reunindo tudo o que venho aprendendo ao longo do curso. Quero que ele sirva como um portfólio sólido para o futuro.

Tecnologias Utilizadas

  • HTML5
  • CSS3 (com Flexbox e Grid)
  • Responsividade para dispositivos móveis
  • Framework (Bootstrap)

Objetivo

Este projeto tem como objetivo aplicar, praticar e consolidar os conhecimentos adquiridos nas aulas de Desenvolvimento Web, servindo como uma base para futuras evoluções e como peça de portfólio profissional..

Próximas melhorias

  • Adicionar interatividade com JavaScript
  • Integrar com um backend simples para simular pedidos
  • Sistema de login para clientes
  • Melhorar responsividade e UX em dispositivos móveis
  • Adicionar animações suaves com CSS

Considerações Finais

Este projeto representa um passo importante na minha jornada como desenvolvedor. Estou animado para continuar evoluindo e aplicando novos conhecimentos a cada atualização!
